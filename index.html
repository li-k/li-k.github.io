Recipedia

With so many different cooking resources and recipes out there, Recipedia is a web application where you can aggregate your favorite recipes. If you're not sure what to make, let the Recipe Generator help you decide!

Recipedia started off as a week-long Express.js/MongoDB project when I was in CODE: Rosie. I continued to use the app long after I initially made it, so in early 2019 I ended up recreating it in Ruby on Rails and PostgreSQL as I'm more familiar with that framework and database.
