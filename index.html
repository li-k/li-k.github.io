Recipedia

With so many different cooking resources and recipes out there, Recipedia is a web application where you can aggregate your favorite recipes. If you're not sure what to make, let the Recipe Generator help you decide!

Recipedia started off as a week-long Express.js/MongoDB project when I was in CODE: Rosie. I continued to use the app long after I initially made it, so in early 2019 I ended up recreating it in Ruby on Rails and PostgreSQL.

I still make the occasional update, such as adjusting for responsive design and adding an autofill feature for new recipes.

Dishlist

Dishlist is a web application to keep track of all the places you want to eat. Adding restaurants is easy with Yelp API integration. As a fun bonus, you can also pick your own ✨color scheme✨.

Similar to Recipedia, Dishlist was a week-long project in CODE: Rosie...and also food-related. Sensing a theme? I haven't updated the code much since, though I would love to eventually revisit it.
